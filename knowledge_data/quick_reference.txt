QUICK REFERENCE GUIDE - VOLLEYBALL DECISION MAKING

SERVING QUICK DECISIONS
- Ahead by 3+: Use jump serve if consistent
- Behind or close: Use float serve
- Set point: Use safest serve
- Need ace: Target weakest passer

ATTACKING QUICK DECISIONS
- Single block: Attack that direction
- Double block: Tip or roll shot
- No block: Power attack
- Off-balance: Tip or roll shot

BLOCKING QUICK DECISIONS
- Predictable setter: Commit to block
- Unpredictable setter: Read and react
- Quick attack: Commit early
- Outside attack: Read first, then commit

DEFENSE QUICK DECISIONS
- Power attack: Play deep
- Tip/roll: Play up
- Unknown: Split the difference
- Block up: Play deep behind

SETTING QUICK DECISIONS
- Outside open: Set outside
- Middle open: Set middle
- Opposite advantage: Set opposite
- All covered: Use back set or dump

MATCH SITUATION QUICK DECISIONS
- Set point for you: Play safe
- Set point against: Play aggressive
- Leading by 5+: Can take risks
- Trailing by 5+: Play safe, minimize errors
- Close game: Use most reliable plays

TIMEOUT QUICK DECISIONS
- Opponent 3+ point run: Call timeout
- Your team 2+ errors: Call timeout
- Before set point: Consider timeout
- Need strategy change: Use timeout

SUBSTITUTION QUICK DECISIONS
- Player 3+ errors: Consider sub
- Unfavorable matchup: Consider sub
- Player fatigued: Consider sub
- Current lineup working: Don't sub

ROTATION QUICK DECISIONS
- Strong rotation: Be aggressive
- Weak rotation: Play safe, side-out
- Close game: Use best rotation
- Big lead: Can experiment

COMMUNICATION ESSENTIALS
- "Mine!" - Taking the ball
- "Help!" - Need assistance
- "Line!" - Attack down line
- "Cross!" - Attack cross-court
- "Tip!" - Expecting tip
- "Up!" - High ball
- "Short!" - Short serve/attack

ERROR PREVENTION
- Don't serve into net on set point
- Don't attack into block when ahead
- Don't over-commit block when behind
- Don't make unforced errors when close
- Don't panic in pressure situations

SUCCESS FACTORS
- Stay calm under pressure
- Trust fundamentals
- Communicate clearly
- One point at a time
- Adapt to situation
- Stay positive
- Know your role
